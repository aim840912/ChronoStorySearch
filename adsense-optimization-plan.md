# AdSense 廣告收入優化計畫（待完成）

## 已完成的廣告配置

| 位置 | 廣告類型 | 設定 |
|------|---------|------|
| 全域底部 | AnchorAd | 始終顯示 |
| 首頁/搜尋卡片列表 | InFeedAd | 每 8 張卡片插入，最多 5 個 |
| 怪物區與物品區之間 | DisplayAd | 兩區都有結果時 |
| 列表底部 | MultiplexAd | 始終顯示 |
| 收藏列表底部 | MultiplexAd | 收藏數 >= 4 時 |
| MonsterModal 掉落列表 | InFeedAd | 物品 >= 10 時，每 10 個插入，最多 3 個 |
| ItemModal 來源列表 | InFeedAd | 來源 >= 10 時，每 10 個插入，最多 2 個 |
| ScrollExchange 分組間 | InFeedAd | 每 3 組插入，最多 3 個 |
| ScrollExchange 底部 | MultiplexAd | 有資料時顯示 |

## 策略 4：搜尋結果前新增 DisplayAd

**視覺佈局：**
```

│  搜尋欄 [__________________] [篩選按鈕]     │
├─────────────────────────────────────────────┤
│  ┌ FilterTabs: 全部 | 怪物 | 物品 ────────┐ │
│  └─────────────────────────────────────────┘ │
│                                              │
│  ╔══════════════════════════════════════════╗ │
│  ║           DisplayAd（橫幅）              ║ │
│  ╚══════════════════════════════════════════╝ │
│                                              │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐        │
│  │Card 1│ │Card 2│ │Card 3│ │Card 4│        │
│  └──────┘ └──────┘ └──────┘ └──────┘        │
```

**觸發條件**：有搜尋詞或篩選條件時才顯示（不影響首頁瀏覽體驗）

### AdSense 政策合規分析
- **完全合規** — 橫幅廣告放在內容區域之間是最常見的合法放置方式
- **風險**：無

### 涉及檔案
- `src/components/lists/AllItemsView.tsx` — 在搜尋結果的 return 區塊頂部加入 `<DisplayAd />`

### 預估收入影響：+10-15%

## 策略 5：SEO 獨立頁面（長期策略）

### 變更內容
為每個怪物/物品建立獨立的 SEO 友好頁面（如 `/monster/faust`、`/item/dark-scroll-for-shield`），每頁有完整的廣告配置

### 概念佈局：
```
┌─────────────────────────────────────────────┐
│  ChronoStory Search — Faust 詳細頁面        │
├─────────────────────────────────────────────┤
│                                              │
│  ╔══════════════════════════════════════════╗ │
│  ║         DisplayAd（頁面頂部橫幅）         ║ │
│  ╚══════════════════════════════════════════╝ │
│                                              │
│  ┌──────────────────────────────────────────┐ │
│  │          怪物資訊區塊                     │ │
│  │  圖片 | 屬性 | 出沒地點                   │ │
│  └──────────────────────────────────────────┘ │
│                                              │
│  ╔══════════════════════════════════════════╗ │
│  ║         DisplayAd（區塊間橫幅）           ║ │
│  ╚══════════════════════════════════════════╝ │
│                                              │
│  ┌──────────────────────────────────────────┐ │
│  │          掉落物品列表                     │ │
│  │  (穿插 InFeedAd)                         │ │
│  └──────────────────────────────────────────┘ │
│                                              │
│  ╔══════════════════════════════════════════╗ │
│  ║         MultiplexAd（頁面底部）           ║ │
│  ╚══════════════════════════════════════════╝ │
│                                              │
│  ╔══════════════════════════════════════════╗ │
│  ║         AnchorAd（固定底部）              ║ │
│  ╚══════════════════════════════════════════╝ │
└─────────────────────────────────────────────┘
```

### AdSense 政策合規分析
- **完全合規** — 獨立頁面是最標準的廣告放置場景
- **額外好處**：SEO 獨立頁面可被 Google 索引，帶來有機流量
- **風險**：無

### 注意事項
- 這是**長期策略**，需要較大的工程投入
- 需要建立動態路由 `app/monster/[slug]/page.tsx` 和 `app/item/[slug]/page.tsx`
- 需要 SSR/SSG 以支援 SEO

### 涉及檔案（新建）
- `src/app/monster/[slug]/page.tsx` — 怪物詳細頁
- `src/app/item/[slug]/page.tsx` — 物品詳細頁

### 預估收入影響：+50-100%（因為帶來全新的有機流量）
