# reqJob 欄位說明

> `reqJob` 是一個位元遮罩 (Bitmask)，用來表示哪些職業可以裝備該物品。

---

## 位元定義

| 位元 | 值 | 職業 |
|------|-----|------|
| 0 | 1 | 劍士 (Warrior) |
| 1 | 2 | 魔法師 (Magician) |
| 2 | 4 | 弓箭手 (Bowman) |
| 3 | 8 | 飛俠 (Thief) |
| 4 | 16 | 海盜 (Pirate) |

---

## 數值解讀

| reqJob | 二進位 | 可用職業 | 數量 |
|--------|--------|---------|------|
| **-1** | 全部 | 通用裝備（特殊） | 7 |
| **0** | 00000 | 新手/全職業 | 164 |
| **1** | 00001 | 劍士 | 334 |
| **2** | 00010 | 魔法師 | 234 |
| **3** | 00011 | 劍士 + 魔法師 | 3 |
| **4** | 00100 | 弓箭手 | 259 |
| **8** | 01000 | 飛俠 | 301 |
| **9** | 01001 | 劍士 + 飛俠 | 11 |
| **13** | 01101 | 劍士 + 弓箭手 + 飛俠 | 6 |
| **16** | 10000 | 海盜 | 82 |

---

## 計算範例

```
reqJob = 9 (二進位: 01001)
= 劍士(1) + 飛俠(8)
= 可被劍士和飛俠裝備

reqJob = 13 (二進位: 01101)
= 劍士(1) + 弓箭手(4) + 飛俠(8)
= 可被劍士、弓箭手、飛俠裝備
```

---

## 程式碼使用範例

```typescript
// 檢查物品是否可被某職業裝備
function canEquip(reqJob: number, jobClass: number): boolean {
  if (reqJob === 0 || reqJob === -1) return true; // 全職業可用
  return (reqJob & jobClass) !== 0;
}

// 職業常數
const JOB = {
  WARRIOR: 1,
  MAGICIAN: 2,
  BOWMAN: 4,
  THIEF: 8,
  PIRATE: 16,
};

// 範例
canEquip(9, JOB.WARRIOR);  // true (劍士可用)
canEquip(9, JOB.MAGICIAN); // false (魔法師不可用)
canEquip(9, JOB.THIEF);    // true (飛俠可用)
```

---

## 備註

- `reqJob = 0` 表示新手或全職業可用（如靴子、手套等通用裝備）
- `reqJob = -1` 是特殊情況，通常是造型裝備
- 海盜數量最少是因為這是後期加入的職業
